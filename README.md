# 数字货币技术分析与AI预测系统

基于Coinglass API的专业级数字货币技术分析工具，集成DeepSeek AI进行智能趋势预测。

## 🚀 主要功能

### 📊 数据获取
- **Coinglass API集成**: 获取实时K线数据（OHLCV）
- **多时间框架**: 支持1分钟到1周的多种时间间隔
- **备用数据源**: 自动切换到CoinGecko API作为备用
- **主流币种**: 支持BTC、ETH、SOL、XRP等主流数字货币

### 📈 技术指标分析
- **移动平均线**: MA5、MA20、MA50
- **相对强弱指标**: RSI(14)
- **KDJ随机指标**: K、D、J三线分析
- **MACD指标**: 快慢线交叉和柱状图
- **布林带**: 上中下三轨分析
- **成交量分析**: 量价关系和成交量比率
- **斐波那契回调**: 23.6%、38.2%、50%、61.8%关键位

### 🤖 AI智能预测
- **DeepSeek AI集成**: 专业技术分析师级别的趋势预测
- **多维度分析**: 技术面、资金面、市场情绪综合分析
- **精准定位**: 支撑位、阻力位、买入卖出区间
- **风险控制**: 止损位建议和风险等级评估
- **信号强度**: 1-10分的信号强度评分

### 📱 用户界面
- **现代化GUI**: 基于Tkinter的直观操作界面
- **多标签页**: 小时级、日级分析和AI预测分离显示
- **实时图表**: 交互式技术分析图表
- **批量对比**: 多币种同时分析对比
- **数据导出**: 支持图表、数据和报告导出

## 🛠️ 安装配置

### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 配置API密钥
复制配置文件模板：
```bash
cp .env.example .env
```

编辑`.env`文件，添加您的API密钥：
```
# Coinglass API密钥（在 https://www.coinglass.com/zh/pro/api 获取）
COINGLASS_API_KEY=your_coinglass_api_key_here

# DeepSeek API密钥（在 https://platform.deepseek.com 获取）
DEEPSEEK_API_KEY=your_deepseek_api_key_here
DEEPSEEK_BASE_URL=https://api.deepseek.com/v1
```

### 3. 运行程序
```bash
# 启动完整GUI程序
python coinglass_analyzer.py

# 或运行功能演示
python run_demo.py
```

## 📖 使用指南

### 基本操作流程

1. **配置API密钥**
   - 在程序界面的"API配置"区域输入密钥
   - 点击"保存配置"按钮

2. **获取数据**
   - 选择要分析的币种（如BTC、ETH）
   - 选择时间框架（1h、4h、1d、1w）
   - 设置数据量（100-1000条）
   - 点击"获取数据"按钮

3. **技术分析**
   - 点击"技术分析"按钮计算所有技术指标
   - 查看右侧的技术指标数值和交易信号

4. **AI预测**
   - 选择预测类型（短期、中期、长期）
   - 点击"AI预测"按钮获取智能分析
   - 查看详细的趋势预测和交易建议

5. **多币种对比**
   - 在"多币种对比"标签页选择要对比的币种
   - 点击"开始对比"进行批量分析

### 图表类型说明

- **价格+MA**: 价格走势和移动平均线
- **布林带**: 布林带通道分析
- **RSI**: 相对强弱指标
- **MACD**: 指数平滑异同移动平均线
- **KDJ**: 随机指标三线分析
- **成交量**: 成交量和量价关系分析
- **综合分析**: 所有指标的综合显示

### 技术信号解读

#### 买入信号 ✅
- MA金叉：短期均线上穿长期均线
- RSI超卖：RSI < 30且开始回升
- KDJ金叉：K线上穿D线且在低位
- MACD金叉：MACD线上穿信号线
- 布林带支撑：价格触及下轨后反弹
- 放量突破：成交量放大伴随价格突破

#### 卖出信号 ❌
- MA死叉：短期均线下穿长期均线
- RSI超买：RSI > 70且开始回落
- KDJ死叉：K线下穿D线且在高位
- MACD死叉：MACD线下穿信号线
- 布林带阻力：价格触及上轨后回落
- 缩量下跌：成交量萎缩伴随价格下跌

#### 关键技术位
- **支撑位**: 斐波那契38.2%、50%、61.8%回调位
- **阻力位**: 前期高点、移动平均线、布林带上轨
- **突破位**: 关键技术位的有效突破确认
- **止损位**: ATR动态止损或关键支撑位下方

## 🎯 技术分析方法

### 经典技术分析理论
1. **道氏理论**: 趋势分析的基础
2. **波浪理论**: 市场波动的规律性
3. **江恩理论**: 时间和价格的关系
4. **斐波那契数列**: 自然回调和扩展水平

### 技术指标组合策略
1. **趋势跟踪**: MA系统 + MACD
2. **超买超卖**: RSI + KDJ
3. **突破确认**: 布林带 + 成交量
4. **支撑阻力**: 斐波那契 + 前期高低点

### AI预测算法
- **多指标融合**: 综合所有技术指标状态
- **模式识别**: 识别经典的技术分析形态
- **概率计算**: 基于历史数据的概率统计
- **风险评估**: 多维度风险因子分析

## 🔧 故障排除

### 常见问题

1. **API密钥错误**
   - 检查密钥是否正确复制
   - 确认API密钥权限设置
   - 验证网络连接状态

2. **数据获取失败**
   - 检查网络连接
   - 验证API额度是否充足
   - 尝试切换到备用数据源

3. **图表显示问题**
   - 确保安装了matplotlib
   - 检查系统字体配置
   - 尝试重新启动程序

4. **AI预测失败**
   - 验证DeepSeek API密钥
   - 检查API余额
   - 确认网络访问正常
